# Sentinel AI

A premium cinematic SaaS landing page built with React 19, Vite 8, TypeScript, and Tailwind CSS v4.

## Features

- **Hero Section** — Two converging hands (robotic + human) in a "Creation of Adam" composition with concentric ring glow animation
- **Cinematic Design** — Dark futuristic aesthetic, liquid-metal text, glassmorphism navbar
- **Responsive** — Fully responsive across desktop and mobile

## Tech Stack

- React 19 + TypeScript
- Vite 8
- Tailwind CSS v4
- No external animation libraries — all CSS keyframe animations

## Getting Started

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
```
